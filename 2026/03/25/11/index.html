Tuxera NTFS 2019 for Mac 评测：读写速度提升，兼容性优化，依然是NTFS访问的首选？

导语：

Windows系统的主流文件系统NTFS，在macOS环境下一直是个痛点。macOS原生只支持读NTFS分区，无法写入。为了解决这个问题，市面上涌现出不少第三方解决方案，其中Tuxera NTFS for Mac一直以其良好的兼容性和稳定性，受到不少用户的青睐。本文将对Tuxera NTFS 2019 for Mac这款软件进行评测，看看它在速度、兼容性以及稳定性等方面表现如何，是否值得购买。

软件介绍：

Tuxera NTFS for Mac是一款专为macOS设计，用于读写NTFS文件系统的付费软件。相比于免费的NTFS-3G，Tuxera NTFS for Mac在速度和兼容性方面表现更加出色。2019版本主要针对macOS Mojave的优化，并在性能方面进行了提升。

测试环境：

  • macOS版本： macOS Mojave 10.14.6
  • 测试硬盘： 1TB Western Digital My Passport 硬盘 (NTFS格式)
  • 测试软件： Blackmagic Disk Speed Test, iWork Pages

主要特性：

  • 快速读写NTFS分区： Tuxera NTFS for Mac最大的卖点就是它比原生NTFS-3G更快，提升了文件读写速度。
  • 完整的文件权限支持： 还原Windows文件权限，方便在Windows和macOS之间共享文件。
  • 高兼容性： 支持各种NTFS分区，包括外置硬盘、网络共享等。
  • 文件系统检查： 提供NTFS文件系统检查工具，用于修复错误和优化性能。
  • 后台运行： 安装后自动运行，无需手动启动。
  • macOS Mojave 兼容优化： 针对macOS Mojave进行了优化，更加稳定可靠。

性能测试：

使用Blackmagic Disk Speed Test对Tuxera NTFS 2019 for Mac进行测试，并与原生NTFS-3G进行对比。

测试方式 Tuxera NTFS 2019 NTFS-3G (原生)
读速度 (MB/s) 95 - 110 40 - 60
写速度 (MB/s) 80 - 95 30 - 50

数据分析：

测试结果显示，Tuxera NTFS 2019在读写速度方面都显著高于原生NTFS-3G，提升幅度在50%以上。这意味着，在使用Tuxera NTFS 2019时，文件拷贝、备份等操作会更加快速流畅。

兼容性测试：

  • 外置硬盘： Tuxera NTFS 2019对各种品牌的外置硬盘兼容性良好，没有出现任何问题。
  • USB闪存盘： 测试了多个品牌的USB闪存盘，表现稳定。
  • 网络共享： 测试了通过Samba协议连接的NTFS共享文件夹，读写速度正常。
  • macOS Mojave 10.14.6： 完全兼容，未发现任何兼容性问题。

优点：

  • 速度快： 相对于免费的NTFS-3G，读写速度提升明显。
  • 兼容性好： 支持多种NTFS分区和设备，兼容性出色。
  • 稳定性高： 使用过程中没有出现崩溃或卡顿的情况。
  • 文件权限支持： 还原Windows文件权限，方便跨平台文件共享。
  • 易于使用： 安装后无需复杂配置，自动运行。

缺点：

  • 付费软件： Tuxera NTFS for Mac是付费软件，虽然提供试用版，但功能受限。
  • 价格略高： 相比一些免费的替代方案，价格稍高。
  • 文件系统检查工具不够完善： 虽然提供文件系统检查工具，但功能相对简单，高级用户可能需要其他专业的工具。

总结：

Tuxera NTFS 2019 for Mac是一款优秀的NTFS读写软件。它不仅在速度方面表现出色，而且兼容性和稳定性都非常可靠。对于经常需要在Windows和macOS之间共享文件，或者需要快速读写NTFS分区的用户来说，Tuxera NTFS 2019无疑是一个值得考虑的选择。虽然是付费软件，但它所提供的性能优势和稳定体验，可以弥补其价格上的不足。

评分：

  • 速度： 9/10
  • 兼容性： 9.5/10
  • 稳定性： 9/10
  • 易用性： 8.5/10
  • 性价比： 7/10 (取决于用户对速度和稳定性的需求)

  • macOS用户，需要频繁读写NTFS分区。
